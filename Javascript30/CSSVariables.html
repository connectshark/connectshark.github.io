# css變數寫法

個人從來沒用過css變數

查了資料才知道目前主流瀏覽器皆有支援css變數寫法

這邊做紀錄

  1. 變數需寫在html下或是:root下
DETAILS

  1. 變數key值前需兩條dash

呈現如下

:root{
  --base:#ffa;
  --spacing: 10px;
  --blur: 10px;
}

# css變數使用

在需要使用的地方用var(key值)

使用如下

.hl{
  color: var(--base);
}
